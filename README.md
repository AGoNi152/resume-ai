# 盛宸亮 - AI 交互式简历

一个精美的翻书式交互简历，内置 AI 问答助手，支持中英文切换。

## 特性

- 📖 **翻书式设计** - 双页展示，桌面端支持翻页导航
- 🤖 **AI 问答助手** - 右下角悬浮按钮，可询问简历相关问题
- 🌐 **中英文切换** - 一键切换语言
- 📱 **响应式布局** - 适配桌面和移动设备
- 🖨️ **PDF 导出** - 支持打印为 PDF

## AI 功能说明

AI 助手支持两种模式：

### 1. 本地知识库（默认）
- 无需配置，开箱即用
- 基于预设的简历信息回答问题
- 适合常见问题（姓名、学校、经历等）

### 2. 在线 AI（需配置 API Key）
1. 点击 AI 助手右上角的 **设置** 按钮
2. 输入 [SiliconFlow](https://siliconflow.cn/) API Key
3. 获得更智能、更自然的回答

**安全提示**：API Key 仅保存在浏览器本地，不会上传到任何服务器。

## 部署方式

### 方式一：静态托管（推荐）

1. **Vercel** (国内访问较慢)
   - 导入 GitHub 仓库自动部署
   
2. **Netlify** (国内访问较慢)
   - 拖拽上传文件夹即可

3. **GitHub Pages** (国内可访问)
   - 推送到 GitHub 仓库
   - 开启 Pages 功能

4. **Cloudflare Pages** (国内可访问)
   - 连接 GitHub 自动部署

5. **Gitee Pages** (国内访问快)
   - 推送到 Gitee 仓库
   - 开启 Pages 服务

### 方式二：国内云服务器

将 `index.html` 上传到任意 Web 服务器即可。
